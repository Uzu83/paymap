# 福岡アウトリーチ文面（MVP・未検証シード前提）

> 用途: 商店街組合・観光案内所・市/FGN 担当への初回連絡。  
> デモ: https://paymap-six.vercel.app · 市向け説明: `/for-city` · FAQ: `/faq`

**共通の正直さ（必ず入れる）**

- Paymap は **MVP（プロトタイプ）** です。福岡の実在店舗は OpenStreetMap 由来のシードを載せています。
- 決済対応は **チェーン一般推定** または **未検証** から始め、現地体験・利用者報告で更新します。
- **虚偽のトラクション（捏造報告・現地確認済みの装い）はしません。** 報告数は `/metrics` で実数のみ公開します。
- 市・支援プログラムへの **採択・公式連携は未確定** です。個人開発プロジェクトです。

## 1. 商店街組合・商工会向け（メール / DM）

**件名案:** 【ご相談】天神〜博多エリアのキャッシュレス可否マップ試験（Paymap・無償）

```
〇〇商店街振興組合
ご担当者様

突然のご連絡失礼いたします。Paymap（使えるペイ）と申します。
「この店で、今自分の PayPay やクレジットが通るか」を地図で確認できる Web アプリを、
福岡・天神〜博多を中心に個人で開発しています。

貴商店街エリアの来街者・店舗様のお役に立てないか、3ヶ月程度の試験導線のご相談をしたくご連絡しました。

■ いま出来ていること（MVP）
・天神〜博多の実在店舗（OpenStreetMap 由来）を地図表示
・PayPay / クレジット等でフィルタ
・来店者が「今使えた / 使えなかった」と報告すると、その店の信頼度が更新される

■ 正直にお伝えすること
・決済情報は多くが「未検証」またはチェーン一般の「推定」です。現地確認済みを装いません。
・ユーザー報告はこれから蓄積する段階で、審査用の数字を捏造しません。
・市や FGN への採択・公式連携はまだありません。個人プロジェクトです。

■ お願いしたいこと（いずれも試験・無償）
・商店街のチラシ・公式 SNS に地図 URL または QR の試験掲載
・パイロット期間中、店舗様へ「報告ボタンの使い方」の周知協力（強制ではありません）
・フィードバック（導線が分かりにくい、店名の扱い等）

デモ: https://paymap-six.vercel.app
市向け1枚説明: https://paymap-six.vercel.app/for-city

15分程度のオンライン説明、またはメールでのご意見でも結構です。
ご多忙のところ恐れ入りますが、ご検討いただけますと幸いです。

Paymap（使えるペイ）
[お名前]

```

## 2. 観光案内所・ホテルフロント向け（短文 DM / チラシ添え）

**想定:** 案内カウンター用の短い依頼。印刷物に QR を載せる前提。

```
【観光案内・ホテルフロント向け】

Paymap（使えるペイ）は、福岡の飲食・小売店で
「PayPay / クレジットが通るか」を地図で確認できる無料 Web アプリです（MVP）。

・天神〜博多の実在店を OpenStreetMap から表示
・未検証の店はバッジで明示（嘘の「確認済み」はしません）
・利用後に「使えた/使えなかった」報告で次の来訪者の参考に

試験として、案内所・フロントのチラシ置き場やデジタルサイネージに
QR（https://paymap-six.vercel.app）の掲載をご検討いただけないでしょうか。
市公式連携ではなく個人開発の試験導線です。掲載期間・撤回はいつでも可能です。

英語の簡易説明は /for-city ページ下部にあります。
ご質問は [連絡先] までお願いします。
```

**カウンター用一言（口頭）:**

> 「キャッシュレスが不安な方向けに、店ごとの対応を地図で見られる試験サイトがあります。未確認の店は『未検証』と出ます。QR はこちらです。」
